# Homenagem à Petz – Linktree

Página simples em HTML, CSS e JS inspirada em páginas do tipo Linktree, criada como homenagem não oficial à Petz.  
A Petz é uma rede brasileira de pet shops com atuação física e digital, oferecendo produtos e serviços para animais de estimação em um ecossistema integrado.

## Sobre o projeto

Este projeto tem como objetivo centralizar, em uma única página, links úteis relacionados à Petz (site, serviços e canais oficiais) em um layout limpo e responsivo.

- Projeto independente, criado apenas como homenagem.
- Não possui vínculo formal, patrocínio ou endosso da Petz.
- Não utiliza logotipo oficial nem elementos proprietários; apenas design autoral.

## Estrutura de arquivos

- index.html – Estrutura principal da página de links.
- style.css – Estilos, cores e layout (inclui tema claro/escuro).
- script.js – Lógica básica para alternância de tema e interações simples.
- assets/ – Pasta opcional para imagens ou ícones autorais.

## Como executar

1. Faça o download ou clone o repositório.
2. Abra o arquivo index.html diretamente no navegador.
3. Edite os links no HTML para apontar para as páginas oficiais da Petz ou outras URLs que desejar.
4. Opcional: publique em serviços como GitHub Pages, Netlify ou Vercel para ter a página online.

## Personalização

- Cores: ajuste as variáveis no início de style.css para combinar com a identidade visual que você deseja, sem copiar identidades protegidas.
- Tipografia: altere a família de fontes em style.css conforme sua preferência.
- Conteúdo: atualize os textos de título, subtítulo e botões para refletir sua mensagem de homenagem.

## Aviso legal

- Esta é uma página não oficial, criada apenas para fins educacionais e de admiração pela marca Petz.
- Todas as marcas, nomes e eventuais referências à Petz pertencem aos seus respectivos detentores de direitos.
- Antes de usar logotipos, imagens oficiais ou outros ativos da Petz, consulte sempre as políticas da empresa e a legislação de direitos autorais aplicável.
